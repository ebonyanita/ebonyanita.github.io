
# Ebony Site (Multi‑Page)

Files:
- index.html — About (home)
- research.html — Research
- conferences.html — Conferences
- teaching.html — Teaching
- cv.html — CV (links to CV_Granada.pdf)
- contact.html — Contact
- style.css — shared minimal styles
- CV_Granada.pdf — your uploaded CV

How to publish with GitHub Pages (project site):
1) Create a public repo (e.g., `predoc-site`).
2) Upload all files above.
3) Repo → Settings → Pages → Source: Deploy from a branch → Branch: main, Folder: /root.
4) Your site will be at https://USERNAME.github.io/REPO-NAME/
5) Add `photo.jpg` to show your headshot (optional, same folder as the HTML files).
